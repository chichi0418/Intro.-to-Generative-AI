## HW1 — Multi-Provider AI Chatbot

A ChatGPT-style chat interface supporting multiple AI providers.

**Features**
- Multi-provider: OpenAI, Anthropic Claude, Google Gemini, xAI Grok
- Streaming responses via SSE
- Conversation history with localStorage persistence
- Adjustable parameters: Temperature, Top P, Max Tokens
- Built-in system instruction presets
- Export conversations as Markdown

**Stack**
- Frontend: Vite + React + TypeScript + Tailwind CSS → GitHub Pages
- Backend: Express.js (Node.js) → Render
